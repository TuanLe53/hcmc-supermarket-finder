## Built With
- Next.js
- PostgreSQL
- Drizzle-ORM
- Tailwind CSS
- shadcn/UI

## Features
- JWT Authentication and Authorization
- Find nearby supermarkets
- Find nearby wishlists
- Accept other user's wishlist
